The short version

Get the work done, without babysitting the tool

Most people looking for a Claude Code alternative want the same thing: get the work done without babysitting the tool.

Wuwei does that. You say what you want, it does the work, and your code never leaves your machine.

Why Wuwei

Four reasons

  • Free, for real. No subscription, no API-key juggling. Wuwei works out of the box. Free is how we think a good tool should start, not a limited-time hook.
  • Runs locally. Your code stays on your computer. Nothing gets shipped off somewhere you can't see.
  • Open and auditable. The code is open source (MIT). You can read exactly what it does. No hidden region tagging, no surprise bans.
  • Model-agnostic. Bring Claude, any OpenAI-compatible endpoint, or a local model. You can leave anytime, which is the whole reason it's safe to stay.

A good tool shouldn't fence you in. Wuwei lets you walk away whenever you want, and that's exactly why you can settle in.
